# 🧠 Focus Trainer

A simple and effective web-based game designed to improve your short-term memory and concentration.

## 🎮 How to Play
1. **Choose Difficulty:** Select how many pairs (3, 4, or 5) you want to memorize from the dropdown menu.
2. **Memorize:** Watch the sequence of character pairs. Each appears for 3 seconds.
3. **Get Ready:** You have a 5-second countdown to prepare after the sequence ends.
4. **Answer:** Type the sequence into the input field within 10 seconds. The game automatically handles uppercase letters.
5. **Check Result:** See your accuracy and enjoy a performance-based cat image!

## 🛠️ Features
- **Dynamic Difficulty:** Choose between 3 to 5 pairs per session.
- **Auto-Formatting:** Input field automatically converts text to uppercase.
- **Statistics:** Tracks your success rate and total sessions played.
- **Responsive Design:** Works on desktops and mobile browsers.

## 🚀 Getting Started
To play the game locally, simply open `index.html` in any modern web browser.

> **Note:** To reset your statistics or change the number of pairs during a session, please **refresh the page (F5)**.
